# IoT-Based IV Fluid Monitoring System

## Overview

A smart healthcare monitoring system designed to track IV fluid levels in real time using a weight-based sensing system with IoT connectivity for remote monitoring and alerts.


## Key Features

* Real-time IV fluid level monitoring
* Weight-based measurement system
* Cloud-based data logging and visualization
* Alert system for critical levels
* Web dashboard for remote monitoring

* C/C++
